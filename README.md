# 💧 MAJUNGMUL (MJM) Global Scholarship Platform
> **"Flowing Talent, Growing Future"**
> 블록체인 기반 성취 보상형 글로벌 장학 SaaS 플랫폼

## 🌟 Overview (개요)
마중물(Majungmul) 프로젝트는 전통적인 기부 방식의 불투명성을 해결하기 위해 설계된 **'성취 보상형(Proof of Achievement)'** 장학 생태계입니다. **MJM 토큰**을 통해 국경 없는 교육 기회를 제공하고 기부금의 흐름을 100% 투명하게 공개합니다.

## ♻️ The Gratitude Loop (보은의 고리)
마중물의 핵심 철학은 수혜자가 성공하여 다시 기여자로 돌아오는 순환 구조에 있습니다. 졸업생들은 시장에서 MJM 토큰을 매수하여 재단에 기부함으로써 다음 세대의 '마중물'이 됩니다.

## 🛠 Tech Stack (기술 스택)
- **Network:** Polygon (L2)
- **Token:** MJM Token
- **Frontend:** HTML5, Tailwind CSS, JavaScript
- **Core Strategy:** Transparent Distribution & Multi-sig Security
